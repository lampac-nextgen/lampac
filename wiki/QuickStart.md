# Быстрый старт (Quick Start)

Это руководство поможет вам запустить сервер **Lampac NextGen**, настроить источники контента и подключить их к приложению Lampa на вашем Smart TV (или другом устройстве) всего за 5 минут.

---

## Шаг 1: Запуск сервера

Самый простой способ развернуть сервер — использовать Docker. Если у вас Linux (Ubuntu/Debian) без Docker, вы можете использовать автоматический скрипт установки.

### Способ А: Docker (Рекомендуемый)
Создайте папку для сервера, перейдите в нее и создайте минимальный файл `docker-compose.yaml`:

```yaml
name: lampac
services:
  lampac:
    image: ghcr.io/lampac-nextgen/lampac
    container_name: lampac
    ports:
      - 9118:9118
    restart: unless-stopped
    volumes:
      - ./init.conf:/lampac/init.conf
```

Рядом с `docker-compose.yaml` создайте файл конфигурации `init.conf` (мы наполним его на Шаге 2).
Затем запустите сервер командой:
```bash
docker compose up -d
```

### Способ Б: Нативная установка на Linux (без Docker)
Просто выполните эту команду от имени пользователя root (или через sudo):
```bash
curl -fsSL https://raw.githubusercontent.com/lampac-nextgen/lampac/main/install.sh | sudo bash
```
*(При такой установке ваш файл настроек будет находиться по пути `/opt/lampac/init.conf`)*.

---

## Шаг 2: Базовая настройка контента

По умолчанию сервер запустится, но чтобы в Lampa появились фильмы, нужно настроить источники и плагины.
Откройте ваш конфигурационный файл `init.conf` и вставьте туда базовый "скелет" настроек:

```jsonc
{
  "listen": {
    "port": 9118
  },
  "online": {
    "name": "Мой Lampac",
    "version": true
  },
  // Настройка предзагрузки плагинов (чтобы не вводить их вручную в ТВ)
  "LampaWeb": {
    "initPlugins": {
      "pirate_store": true,
      "online": true,
      "sisi": true
    }
  },
  // Включение популярных и стабильных VOD провайдеров
  "Kodik": {
    "enable": true,
    "priority": 1
  },
  "Collaps": {
    "enable": true,
    "priority": 2
  },
  "HDVB": {
    "enable": true,
    "priority": 3
  }
}
```
*Как только вы сохраните файл, Lampac автоматически применит настройки. Перезагружать сервер не нужно!*

---

## Шаг 3: Проверка и использование (Интерфейс Lampa)

В Lampac NextGen встроен модуль **LampaWeb**, который позволяет открывать клиент Lampa прямо с вашего сервера.

Откройте браузер на вашем ПК, телефоне или Smart TV и перейдите по адресу:
👉 `http://IP-АДРЕС-ВАШЕГО-СЕРВЕРА:9118`

Если всё сделано верно, вы увидите знакомый интерфейс Lampa. Благодаря настройке `LampaWeb.initPlugins` из шага 2, все необходимые плагины (для просмотра онлайн-кино и контента 18+) уже будут загружены автоматически.

🎉 **Готово!** Теперь при открытии карточки фильма вы увидите кнопку "Онлайн", при нажатии на которую начнется поиск видео через ваш собственный сервер Lampac.

## Что дальше?

Вы прошли "счастливый путь". Lampac NextGen обладает огромным функционалом, который вы можете изучить далее:
- **Защита сервера:** Закройте доступ от посторонних с помощью [WAF (Брандмауэра)](Configuration.md#4-защита-и-безопасность-waf) или настройте приватный доступ через [Аутентификацию accsdb](Configuration.md#5-аутентификация-accsdb).
- **Авторизация через Telegram:** Надоели пароли? Настройте [TelegramAuth и бота](Modules.md#community-telegramauth-и-telegramauthbot) для удобного входа.
- **Торренты:** Хотите смотреть торренты напрямую без скачивания? Изучите встроенный [TorrServer](Modules.md#состояние-модулей-по-умолчанию) и агрегатор раздач [JacRed](Modules.md#состояние-модулей-по-умолчанию).
- **18+ Контент:** Откройте для себя гибкий парсинг сайтов для взрослых через конфигурации [NextHUB](Modules.md#nexthub-18).
- **Обход блокировок и JS-защит:** Если некоторые провайдеры (например, Cloudflare) не отдают видео, настройте [Playwright (Браузерная автоматизация)](Configuration.md#6-playwright-обход-js-защит-cloudflare).
- **Проксирование постеров:** Ускорьте загрузку обложек, настроив [TMDB Прокси и Cub Прокси](Configuration.md#11-дополнительные-технические-настройки).
